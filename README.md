# PADRL-Fusion-Dataset

This dataset supports the research presented in the paper "Cognition-Driven Deep Reinforcement Learning for Robust Multisensor Fusion in Unmanned Inspection Systems." It was collected using an unmanned ground vehicle (UGV), and it contains a total of 8 sequences.

## Dataset Overview

- Platform: UGV (see `UGV.png`)
- Number of sequences: 8
- Trajectory overview: `Trajectory.png`

## Description

The dataset is designed for robust multisensor fusion and autonomous inspection scenarios. Each of the 8 sequences captures a different environmental condition and navigation challenge encountered by the UGV.

## Figures

- `UGV.png`: shows the UGV platform used for data collection.
- `Trajectory.png`: presents the full trajectory map across all 8 sequences.

## Challenges

The trajectory figure includes annotated highlights for the key challenges encountered in each sequence. These challenge annotations indicate:

- difficult terrain segments
- sensor occlusions or interference
- complex maneuvers
- dynamic obstacles and scene changes

The annotations in `Trajectory.png` help illustrate the variability of the dataset and the types of scenarios the UGV faced during data collection.

## Usage

This dataset is intended for experiments in sensor fusion, localization, mapping, and autonomous navigation with a UGV.
